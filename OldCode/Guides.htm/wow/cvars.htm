iterami/Guides.htm/World of Warcraft/CVars
  • CVar names are not case sensitive.
CVar Default Value Recommended Value Notes
alwayscompareitems 0 1
  • If set to 1, hovering over an item in your backpack that can be equipped will display your currently equipped item in that slot.
  • If set to 0, you must hold shift to display item comparisons.
autolootdefault 0 1
  • If set to 1, items are automatically added to your backpack when looting.
  • If set to 0, you must click on items in the loot window to add them to your backpack.
autolootrate 150 100 Number of milliseconds between each item in the loot window being autolooted and added to your backpack.
colorchatnamesbyclass 0 1
  • If set to 1, player names in chat are colored by their class, thus giving you free additional information.
  • If set to 0, player names all have the same color.
enablefloatingcombattext 0 1
  • If set to 1, floating text will be displayed near the center of your screen when your character's state or stats change. Additional CVars are available to customize which changes should be displayed.
  • If set to 0, no changes to your character will be displayed as text floating near the center of your screen.
miniworldmap 1 1
  • If set to 1, your world map is displayed in a regular-sized UI window, allowing you to retain visibility and full character control.
  • If set to 0, your world map takes up the entire screen and obscures your vision. You have keyboard control of your character, but not mouse control.
missingtransmogsourceinitemtooltips 0 1
  • If set to 1, hovering over an item that can be used for transmog will display if you have collected that appearance but not from that specific item. This can be useful as the names of items you use for transmog can be collected as well.
  • If set to 0, hovering over an item will only display a notice if you haven't collected the appearance.
nobuffdebufffilterontarget 0 1
  • If set to 1, displays all buffs and debuffs on your target, which can help you make better decisions during combat.
  • If set to 0, only your own buffs and debuffs will be displayed.
statustext 0 1
  • If set to 1, health and resource bars will display text values instead of only being a solid-color bar. Specific display text is hanlded by the "statustextdisplay" CVar.
  • If set to 0, health and resource bars will not display any text values.
statustextdisplay "NONE" "NUMERIC"
  • If set to "BOTH", health and resource bars will display a percentage between 0% and 100% on the left and remaining value on the right. No maximum is displayed.
  • If set to "NONE", health and resource bars will not display any text values.
  • If set to "NUMERIC", health and resource bars will display "remaining / maximum". No percentage is displayed.
  • If set to "PERCENT", health and resource bars will display a percentage between 0% and 100%. No numbers indicating remaining or maximum are displayed.
xpbartext 0 1
  • If set to 1, always displays text values on your XP bar, including when displaying reputation or honor.
  • If set to 0, values are only displayed when your character info window is open.
